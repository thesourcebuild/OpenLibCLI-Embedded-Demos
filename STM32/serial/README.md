# STM32 Serial Demo

This directory contains the STM32 serial (UART) demo for OpenLibCLI.

## Table of Contents

- [Target](#target)
- [Development Environment](#development-environment)
- [Notes](#notes)

## Target

- Board: STM32 Nucleo-F103RB with RAM=20KB, ROM(Flash)=64KB/128KB
- Transport: Serial UART

## Development Environment

- Keil MDK: 5.43
- IAR EWARM: 9.70.4
- STM32CubeIDE: 2.1.1

## Notes

The project can be opened and built with any of the supported IDE/project files in this folder. The versions above were used when this demo was prepared. The previous versions of the IDEs will also work but you have to integrate the OpenLibCLI.
